Mes modèles enregistrés

Bibliothèque de modèles IA

Catalogue Freev : recherche rapide, compatibilité indicative et installation guidée pour Ollama et LM Studio. Les commandes d’installation sont incluses dans le guide rapide.

Détection de ton appareil en cours...

1. Installer Ollama

Installe Ollama une seule fois. Ensuite, toutes les commandes ollama pull des fiches fonctionneront.

Windows — PowerShell

irm https://ollama.com/install.ps1 | iex

macOS / Linux — Terminal

curl -fsSL https://ollama.com/install.sh | sh

Télécharger un modèle ensuite :

ollama pull qwen3:8b

Pour discuter : ollama run qwen3:8b.

2. Installer LM Studio CLI

L’application LM Studio inclut la commande lms. Pour une installation terminal sans interface, installe llmster, qui fournit aussi lms.

Windows — PowerShell

irm https://lmstudio.ai/install.ps1 | iex

macOS / Linux — Terminal

curl -fsSL https://lmstudio.ai/install.sh | bash

Vérifier puis télécharger un modèle :

lms --helplms get "Qwen3 8B"
Après téléchargement : ouvre Mes modèles installés, sélectionne ton modèle, puis Freev le configure automatiquement.
Runtime :
Essais rapides : Utilise Entrée pour choisir une suggestion.

Les commandes sont fiables pour les fiches officielles ; une variante GGUF exacte s’installe plutôt dans LM Studio.

La bibliothèque distingue les modèles officiels et les formats locaux. Les tailles et la RAM sont des estimations : vérifie toujours la fiche du dépôt avant de télécharger.
